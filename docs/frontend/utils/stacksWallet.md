# stacksWallet.js

`stacksWallet.js` wraps native Stacks wallet connection and transaction helpers.

Public helpers:

- `connectStacksWallet()`
- `disconnectStacksWallet()`
- `getAddresses()`
- `callContract(options)`
- `transferStx(recipient, amount, memo)`
- `isConnected()`
- `getSession()`

Source file: `frontend/src/utils/stacksWallet.js`.
